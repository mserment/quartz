# Damaldr 
![[Damaldr.jpg]]

#lieu #village #hauteTrahison 

Village situé dans les forets dense de l'arriere pays. L'hiver y est rude

# Description
- Petit village
- Jarl : [[ylva|Ylva La louve gardienne]]
- Situation : Alliance avec le village de [[Chathaldur]]

# Particularités
- On peux y voir a l'entré du village une statue d'une femme triomphante avec un loup a ces cotés (Il s'agit du Jarl [[ylva|Ylva La louve gardienne]])

# Necro gameplay
> A adapter pour une utilisation dans ironsworn


*prerequis* :
	Competence en guerisons
	Rituel d'appartenance avec la mort reussi //accomplir un rituel dangereux ou il faut ce liee a une petite creature

*personage* : 
	*les plus* :
	- vivant ou mort
	- Tout type de classe
	- Tout type de race / sexe

*les moins* :  
	- ne peux pas porter de grosse armure (meme en guerrier)
	- est mal vu par les entités du bien
	- si vivant, le personnage n'as pas de malus en general mais a des malus de necro
	- si mort, le personnage a des malus en general mais a des bonus en necro 


*competences de bases*:
	necromant: 
	tous les 3 niveaux les couts en mana sont reduit 
	La nuit les pouvoirs sont plus fort / le jour les pouvoirs sont reduits

pool d'ames	:	
Le pool d'ames est une reserve d'ames a disposition du necromant
Lorsque le necromant est sur le point de tuer un individu / monstre il peux recuperer l'ame de la victime en question

necromant vivant :
Le lien qu'il a avec les morts est faible.
Le necromant est limité sur certains aspects comme la puissance, la zone,la durée des sorts. Certains sorts sont atténuée d'autres sont impossible a utiliser sans sacrifice d'ames (vivant ou emagasiner dans le pool)

necromant mort :	
Le necromant peux ce sacrifier lui meme en offrant sont ame pour devenir plus fort dans tous les domaines des necromants.
Il n'est plus obligatoire d'utiliser des sacrifices dans certains rituel ou pour augmenter sa force.


## Gestion par niveau

niveau 1:
	cause du deces: physique
	liaison demoniaque: petite creature liée du necromant. doit etre tué par le necromant pour debloquer gestion des ames, ne peux pas combatre
	appel de la mort : il aime voir du monde mort (pas de bonus a ce niveau)
	vision de mort : faible, une seule cible, porté courte (peux toucher la cible pour plus de puissance)

niveau 2:
	zone de putrefaction: faible zone,faible durée, impact faible
	reanimation : petite creature, durée faible
	odeur de mort : faible,suit le necromant, pas de portée. 
	porteur funeste : capacité faible

niveau 3:
	doigt de la mort : degat faible, porté faible, doit avoir la cible en vue.
	invocations temporaire: petite creature
	marque de mort: augmentation faible, portée courte
	sacrifice des faibles: degat faible, recuperation pv faible.
	vision de mort: jusqu'a 3 cibles porte courte (peux toucher une cible pour plus de puissance)

niveau 4:
	zone de putrefaction: zone moyenne, durée moyenne, impact moyen
	reanimation: moyenne creature, durée moyenne
	odeur de mort: peut etre lancé a portée faible. peux passer a travers des objets simple (poste classique), durée moyenne

niveau 5:
	cause du deces: physique + mentale
	liaison demoniaque: creature moyenne, doit etre tué par le necromant, necessite une epreuve pour domination creature
	doigt de la mort: degat moyen, portée moyenne, doit avoir la cible en vue.
	invocations temporaire : petite creature x2, durée faible / moyenne creature x1, durée faible
	collecteurs os:  activation de la competence
	porteur funeste : capacité moyenne


### Competences:
lames sort // si vous utiliser une arme en meme temps que vous caster un sort dans l'autre mains, votre attaque fait plus de degat
detection des morts // vous etes capables de sentirs les morts environents (niveau pour portée et precision)
porteur funeste // sac virtuel dans lequel le necromant peux reunir des objets de necromants uniquement
collecteurs os // peux depecer des corps, s'il parvient a reunir les 11 elements d'un corp du meme type (humaniode,creature,etc) peux faire une liaison demoniaque (necessite un gros rituel)
consecration impie // vous pouvez utiliser vos 2 mains pour lancer un sorts (touts les effets sont doublé, couts doublé)
sacrifice des faibles // inflige des degats a une creature invoquée pour recuperer de la vie / mana (voir niveaux)
appel du maitre // les creatures invoquées retourne vers leur maitre quel que soit la distance. (voir niveaux)
sacrifice du maitre // vous pouvez sacrifier de la mana ou des pv pour redonner de la force a vos invocations
appel de la mort // les morts autour de lui l'enivre et il deviens plus fort (gros bonus s'il tue lui meme, durée augmenter avec le niveau)
cause du deces // connais suffisament la mort pour connaitre l'origine d'un deces (physique puis autre)
gestion des ames // le necromantien garde les ames (voir emprisonement des ames) et peux les utiliser plus tard (dmg augmenté, plus de regeneration, xp augmenter etc)
emprisonement des ames // sort qui permet d'emprisonner l'ame des creatures cible. lorsqu'elles meurrent le necromant peut utiliser leurs ames (la recuperation des ames a une chance de reussir)
reanimation // reanimation d'un corp durée et niveau en fonction du niveau du necromant
reanimation forte // reanimation et buff d'un corp. il est plus puissant que vivant pendant une courte durée
invocation armure // invocation d'une armure magique demoniaque autour de vous (durée selon niveau)
invocations temporaire // invocation dans le temps, peux mourrir (permet d'invoquer des objets ou des creatures, porté augmenter avec les niveaux)
invocations permanente // pas de limite de temps, peux mourrir (invocation plus complexe que l'invocation temporaire pour le meme objet / creature)
liaison demoniaque // invocation permanente qui ne peux mourrir (cd quand mort)
zone de putrefaction // profane une zone ce qui augmente les pouvoirs du necromant dans cette zone (empeche les enemies de ce soigner / regen avec le niveau)
doigt de la mort // rayon envoyé sur une cible qui subit des degats mentales
odeur de mort // disperce une odeur de mort . elle affaibli les cibles 
vision de mort // la cible fait face a des allucinations morbides
marque de mort // la cible est marqué et tout les degats contre elles sont augmenter
rituel de tenebres // rituel durant un moment qui permet de renforcer les pouvoirs du necromantien pour une durée determiner
pleine lune // obsurcie le ciel augmentant le pouvoir du necromantien.
teleportation nocturne // vous pouvez vous teleporter de quelques metres (utile en combat), vous devez pouvoir voir l'emplacement de destination
